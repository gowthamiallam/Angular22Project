Definition of Directive :

Directive has some functionality written and it will do

  • create new element
  • modify new element
  • modify the existing behaviour of element also
  • When we you want to go for dynamic class go for ng class
    class="p-4 bg-success" or class="p-4 bg-danger" will come when you inspect it means class is changing dynamically on clicking
  • when ever you want to go for dynamic css gor for ng style
    style="background-color: red; height: 200px; width: 200px;" or style="background-color: green; height: 200px; width: 200px;" will come when you inspect it means style is changing dynamically on clicking
